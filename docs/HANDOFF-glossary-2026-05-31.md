## 3. 关键技术点 / 踩过的坑(本次)

- **Web Speech 受 autoplay 门控**:`speechSynthesis.speak()` 无用户手势会 `error: not-allowed` 静默失败。Chrome 的 **Media Engagement Index** 按域名累积互动分,用多了会放行 → 表现为「时好时坏 / 修了又复发」,且在养高的浏览器里复现不出。**任何 TTS 自动播都不能赌成功,必须有手势兜底。**
- **类名命名空间**:造句游戏已占用全局 `.ab-*`(带 `#666` 背景)。新功能起名要避让(本次用 `qab-*`),否则样式串味(标签变暗方块)。
- **复习/测验例句 = 纯 TTS**,不是录音片段(录音片段是卡片上单独的 🎙 跟读按钮)。
- **排序实现**:重排 `.terms` grid 里的 `.term` DOM,读 `review_stage`;`+加词` 按钮保持末尾;原序用首次排序前的快照(WeakMap)还原;同分用原序兜底。排序状态是会话内的(刷新回原序),各分类独立。
- **全部单词去重**:同词跨课只留一张,取最高完成度那个实例;每次点进去实时重建,保证刚复习完的进度最新。
