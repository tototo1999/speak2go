3. 关键技术点 / 坑

  • Web Speech 受 autoplay 门控:无手势 speak() 会 not-allowed 静默失败。Chrome 按域名累积 Media Engagement Index,用多了放行 → 「时好时坏 / 修了又复发」,养高的浏览器复现不出。TTS 自动播不能赌成功,必须手势兜底。
  • 类名命名空间:造句已占全局 .ab-*(带 #666 背景),新功能要避让(本次 qab-*),否则标签变暗方块。
  • 复习/测验例句 = 纯 TTS,不是录音片段(录音是卡片 🎙 跟读)。
  • 排序:重排 .term DOM 读 review_stage;+加词 留末尾;原序用首次排序前 WeakMap 快照还原;会话内有效、各分类独立。
